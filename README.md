\# 🚀 SpaceX Exploration



> An interactive web experience to explore SpaceX missions, rockets, and launches.









\*\*\*



\## ✨ Features



\- 🛸 Browse SpaceX rockets and mission details

\- 📅 Explore past and upcoming launches

\- 🌍 Responsive design — works on all screen sizes

\- ⚡ Fast and lightweight — no frameworks, pure JS



\*\*\*



\## 🛠 Tech Stack



| Technology | Purpose |

|---|---|

| HTML5 | Page structure and semantics |

| CSS3 | Styling, layout, animations |

| JavaScript (ES6+) | DOM manipulation, logic |

| SpaceX API \*(optional)\* | Live launch data |



\*\*\*



\## 🚀 Getting Started



\### Clone and run locally



```bash

git clone https://github.com/lioubiarabi/spaceX\_exploration.git

cd spaceX\_exploration

```



Then simply open `index.html` in your browser — no server needed.
